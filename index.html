Vanilla Weather App

The Weather App is a web application that allows users to search for weather conditions of a specific location or retrieve the weather information based on their current location. The app utilizes the OpenWeatherMap API to fetch real-time weather data, including temperature, humidity, wind speed, and a brief description of the weather condition. Additionally, it provides a 6-day weather forecast with daily maximum and minimum temperatures.
